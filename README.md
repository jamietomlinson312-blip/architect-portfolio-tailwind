# Architectural portfolio tailwind website

## Overview

This project is a multi-page portfolio website consisting of:

- Landing (Hero) page
- Project Gallery
- Contact Page
- Mobile-friendly layout
- Modern, clean design

The goal of this project is to practice building a polished website, while following good project structure and version control practices, using exclusively tailwindcss (v4) for the styling. In particular, CSS flex boxes and grids were used throughout to place elements precisely and ensure consistent spacing. Additional colours and fonts were added using the @theme directive in the input.css file.

## Built With

- HTML5
- Tailwind CSS
